# Customer Shopping Behavior Analysis

This repository contains a comprehensive data analysis project exploring customer shopping behavior. The project involves data cleaning, SQL-based data exploration and analysis, and visualization through a Power BI dashboard.

## Project Structure

- `dataset/customer_shopping_behavior.csv`: The raw dataset used for analysis. It contains information such as customer demographics (Age, Gender, Location), purchase details (Item, Category, Amount, Size, Color, Season), and behavioral metrics (Review Rating, Subscription Status, Shipping Type, Discount Usage, Previous Purchases, Frequency of Purchases).
- `cleaning_prep.ipynb`: A Jupyter Notebook dedicated to data cleaning and preprocessing. This step ensures the data is ready for accurate analysis.
- `Data_views.sql`: A collection of SQL queries answering key business questions and extracting insights from the cleaned dataset.
- `Customer_Behavior_DashBoard.pbix`: A Power BI dashboard providing an interactive visual overview of the dataset and analysis results.

## SQL Analysis (`Data_views.sql`)

The SQL script contains queries designed to extract the following insights:
1.  **Revenue by Gender:** Total revenue generated by male vs. female customers.
2.  **Discount Usage and Spend:** Identifying customers who used a discount but still spent above the average purchase amount.
3.  **Top Rated Products:** The top 5 products with the highest average review ratings.
4.  **Shipping Type Comparison:** Comparing average purchase amounts between Standard and Express Shipping.
5.  **Subscription Impact:** Analyzing average spend and total revenue between subscribers and non-subscribers.
6.  **Discounted Products:** Identifying the top 5 products with the highest percentage of discount usage.
7.  **Customer Segmentation:** Categorizing customers into 'New', 'Returning', and 'Loyal' based on their number of previous purchases.
8.  **Top Products per Category:** Identifying the top 3 most purchased products within each category.
9.  **Repeat Buyers & Subscriptions:** Analyzing the likelihood of repeat buyers (more than 5 previous purchases) to be subscribed.
10. **Revenue by Age Group:** The revenue contribution broken down by age group.

## Data Cleaning & Preparation (`cleaning_prep.ipynb`)

This notebook covers the initial steps to prepare the data for analysis, which may include handling missing values, standardizing formats, and creating new derived features for deeper analysis (like age groups or specific customer segments).

## Power BI Dashboard (`Customer_Behavior_DashBoard.pbix`)

The Power BI file provides an interactive visual representation of the findings. It allows stakeholders to explore the data dynamically, filtering by demographics, product categories, or time periods to uncover deeper insights into customer behavior.

## How to use

1.  Review the data cleaning steps in `cleaning_prep.ipynb`.
2.  Explore the SQL queries in `Data_views.sql` to understand the data extraction logic.
3.  Open `Customer_Behavior_DashBoard.pbix` in Power BI Desktop to interact with the visualizations.
